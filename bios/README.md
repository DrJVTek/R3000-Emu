# BIOS

Placez ici votre BIOS PS1.

## Nom recommandé

- `bios/ps1_bios.bin`

## Lancer l’émulateur

- Sans arguments: l’émulateur tente de booter automatiquement `bios/ps1_bios.bin` (ou `bios/bios.bin`).
- Sinon: vous pouvez forcer un BIOS avec `--bios=<path>`.

> Note légale: ne versionnez pas un BIOS si vous n’avez pas les droits.
